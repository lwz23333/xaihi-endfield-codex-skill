# Xaihi Endfield Codex Skill

赛希（Xaihi / 塞拉菲娜·赛希）角色 Skill，面向 Codex 的角色扮演与角色知识调用。

本 Skill 基于公开资料、公开剧情语料与可追溯来源进行整理和行为蒸馏，用于模拟赛希的表达风格、知识边界、世界观锚点和 Baker 特殊交互。
