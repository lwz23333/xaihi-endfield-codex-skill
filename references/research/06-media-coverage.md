## 未覆盖媒体

- 游戏客户端内所有实机剧情、个人支线、活动剧情的完整原文；本轮只覆盖补充任务、Baker 和干员档案条目。
- 官方社媒原始角色介绍帖，如森空岛/微博/YouTube/X 的所有原帖与评论上下文。
- 角色所有语音音频的声线细节，只覆盖公开文本整理。
- 2026-06-08 后发布的新剧情、新活动、新访谈、新设定修订。
- 未公开或需登录/游戏进度解锁的资料。

## 主线与衍生材料权重

权重排序：

1. 官方站公告与官方角色介绍搬运，用于版本边界和基础设定。
2. 游戏内档案/台词整理，用于性格、表达、关系和关键场景。
3. 萌娘百科、BWIKI、英文 Wiki 等社区资料，用于交叉验证与补缺。
4. 攻略站战斗机制，用于确认玩法身份，不直接支撑人格。
5. 社区讨论、低质量攻略、未署名搬运不作为核心来源。

## 冲突记录

- 本地 CSP 统一检索入口 source_search.py 在 Windows 下失败，错误为 PermissionError: WinError 5。专用 moegirl_api.py 搜索与 full 模式可用，intro/wikitext 模式遇到 SSL EOF。该失败已记录为 S03、S04。
- 部分攻略站对赛希战斗属性或强度描述可能随版本变动，且有站点呈现字段疑似过时或错误。因此战斗结论以 S05、S11、S13 的交叉项为参考，不进入核心人格。
- 官方角色介绍的一手 skland 页面未直接抓取到，使用 GamerSky 搬运页，页面标注“来源：skland 作者：官方”。此项作为 official_repost 处理，低于官网直链，高于普通社区摘要。
- `mzh.moegirl.org.cn` 移动 URL 可作为指定条目记录；浏览时直接读取不稳定，同条目内容通过萌娘百科可读镜像交叉查看。更新中只吸收与既有资料互相支持的气质和表达信息。
- GitHub 剧情语料库本地副本用于补充公开剧情材料，并保留语料库 README 对 Warfarin Wiki 来源的说明。Baker 材料用于补充赛希日常交互，不改写未覆盖主线事实。
